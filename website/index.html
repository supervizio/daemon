supervizio

PID1-capable process supervisor for containers and Unix systems

Version {{VERSION}}
Get Started View on GitHub

Multi-Service Management

Start, stop, and monitor multiple processes with dependency ordering and graceful shutdown.

Health Checks

HTTP, TCP, and command-based health checks with configurable intervals and retries.

Restart Policies

Flexible policies: always, on-failure, never, unless-stopped. With exponential backoff.

Log Rotation

Automatic rotation by size, age, and count. Optional gzip compression.

PID 1 Mode

Proper init behavior: zombie reaping, signal forwarding, clean shutdown.

Multi-Platform

Linux, FreeBSD, OpenBSD, NetBSD, DragonFlyBSD, and macOS support.

Quick Start

# Install (Debian/Ubuntu)
curl -fsSL {{BASE_URL}}/gpg.key | sudo gpg --dearmor -o /etc/apt/keyrings/supervizio.gpg
echo "deb [signed-by=/etc/apt/keyrings/supervizio.gpg] {{BASE_URL}}/apt stable main" \
  | sudo tee /etc/apt/sources.list.d/supervizio.list
sudo apt update && sudo apt install supervizio
# Create config
cat > /etc/supervizio/config.yaml <<EOF
version: "1"
services:
  - name: myapp
    command: /usr/bin/myapp
    restart:
      policy: always
EOF

# Run
supervizio --config /etc/supervizio/config.yaml
