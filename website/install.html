Installation

Choose your platform and installation method.

Package Repository

Recommended for Linux systems. Packages are GPG-signed.

Debian / Ubuntu / Devuan

# Add GPG key
curl -fsSL {{BASE_URL}}/gpg.key | sudo gpg --dearmor -o /etc/apt/keyrings/supervizio.gpg

# Add repository
echo "deb [signed-by=/etc/apt/keyrings/supervizio.gpg] {{BASE_URL}}/apt stable main" \
  | sudo tee /etc/apt/sources.list.d/supervizio.list

# Install
sudo apt update && sudo apt install supervizio

Rocky Linux / RHEL / Fedora

# Add GPG key
sudo rpm --import {{BASE_URL}}/gpg.key

# Add repository
sudo curl -fsSL {{BASE_URL}}/rpm/supervizio.repo -o /etc/yum.repos.d/supervizio.repo

# Install
sudo dnf install supervizio

Alpine Linux

# Add signing key
sudo wget -O /etc/apk/keys/supervizio.rsa.pub {{BASE_URL}}/apk/supervizio.rsa.pub

# Add repository
echo "{{BASE_URL}}/apk/v3.21/main" | sudo tee -a /etc/apk/repositories

# Install
sudo apk add supervizio

Binary Download

For all platforms including BSD and macOS.

Automatic (Linux/macOS/BSD)

# Detect platform and download
OS=$(uname -s | tr '[:upper:]' '[:lower:]')
ARCH=$(uname -m | sed 's/x86_64/amd64/;s/aarch64/arm64/;s/armv7l/arm/;s/i686/386/')

curl -fsSL "{{GITHUB_URL}}/releases/latest/download/supervizio-${OS}-${ARCH}" -o supervizio
chmod +x supervizio
sudo mv supervizio /usr/local/bin/

Package Files

Download individual packages from releases:

# Debian/Ubuntu (.deb)
sudo dpkg -i supervizio_*.deb

# Rocky/RHEL (.rpm)
sudo rpm -i supervizio-*.rpm

# Alpine (.apk)
sudo apk add --allow-untrusted supervizio-*.apk

From Source

Requires Go 1.25+

git clone {{GITHUB_URL}}.git
cd daemon/src
go build -o supervizio ./cmd/daemon
sudo mv supervizio /usr/local/bin/

Post-Install

Enable Service (systemd)

sudo systemctl enable --now supervizio

Enable Service (OpenRC)

sudo rc-update add supervizio default
sudo rc-service supervizio start

Enable Service (BSD rc.d)

sudo sysrc supervizio_enable=YES
sudo service supervizio start

Verify Installation

supervizio --version
