# Website

## Purpose

Static documentation website for superviz.io project. Hosted on GitHub Pages.

## Structure

```text
website/
├── index.html      # Landing page
├── install.html    # Installation guide
├── config.html     # Configuration reference
└── style.css       # Shared styles
```

## Key Files

| File | Description |
|------|-------------|
| `index.html` | Project overview and quick start |
| `install.html` | Platform-specific installation |
| `config.html` | YAML configuration reference |
| `style.css` | CSS styles (dark theme) |

## Conventions

- Pure HTML/CSS (no JavaScript frameworks)
- Dark theme consistent with terminal aesthetic
- Mobile-responsive design
