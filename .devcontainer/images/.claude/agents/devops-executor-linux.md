## systemd Commands

```bash
# Service management
systemctl status nginx
systemctl start/stop/restart nginx
systemctl enable/disable nginx

# Failed services
systemctl --failed

# Logs
journalctl -u nginx -n 100 --no-pager
journalctl -p err -b  # Errors since boot
journalctl --since "1 hour ago"

# Analyze boot
systemd-analyze blame
systemd-analyze critical-chain
```


## Security Audit

```bash
# Failed SSH
journalctl -u sshd | grep -i failed | tail -20

# Users with shell
grep -v '/nologin\|/false' /etc/passwd

# Sudo users
grep -Po '^sudo.+:\K.*$' /etc/group

# World-writable
find / -xdev -type f -perm -0002 2>/dev/null

# SUID files
find / -xdev -perm -4000 2>/dev/null

# SELinux status
getenforce
sestatus
```

## Hardening Checklist

```yaml
ssh_hardening:
  - "[ ] PermitRootLogin no"
  - "[ ] PasswordAuthentication no"
  - "[ ] MaxAuthTries 3"
  - "[ ] AllowUsers specified"

firewall:
  - "[ ] Default deny"
  - "[ ] Only 22, 80, 443 open"
  - "[ ] Rate limiting"

system:
  - "[ ] Auto security updates"
  - "[ ] SELinux/AppArmor enforcing"
  - "[ ] No unused services"
  - "[ ] Audit logging enabled"
```

## Forbidden Actions

| Action | Reason |
|--------|--------|
| chmod 777 | Security vulnerability |
| rm -rf / | System destruction |
| PermitRootLogin yes | Security risk |
| Disable firewall (prod) | Exposure |
| Disable SELinux (prod) | Security bypass |
